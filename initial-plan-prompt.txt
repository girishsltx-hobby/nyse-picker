Build a LOCAL web application that tracks 9 NYSE tickers during US market sessions (Pre-market, Regular, After-hours) and displays a dashboard with TradingView-style indicators and signals.

========================================================
1) Core Goal
========================================================

- Aggregates to 5-min candles
- Computes indicators used in TradingView (matching TradingView definitions as closely as possible)
- Visualizes on TradingView charts

========================================================
3) Indicator/Signal Requirements (computed on 5-min candles)
========================================================
Compute the following per ticker:

A) EMA crossover
- EMA 9 and EMA 21
- signal: bullish when EMA9 > EMA21, bearish when EMA9 < EMA21
- detect crossover events and timestamp them

B) VWAP (session-aware)
- Compute VWAP separately per session:
  - pre-market session
  - regular session
  - after-hours
- Determine:
  - price above/below VWAP
  - distance to VWAP
  - moving toward VWAP vs moving away from VWAP
  - track slope of VWAP

C) Daily Trend
- Determine bullish/bearish/sideways using:
  - higher highs/higher lows (bullish)
  - lower highs/lower lows (bearish)
  - otherwise neutral
- Provide also an ATR-based trend strength metric (optional)

D) Volume Profile (session profile)
- Compute session volume profile using 5-min candles (approximate is acceptable)
- Identify POC (Point of Control) per session
- Show POC as a line on chart and value in dashboard

E) Support and Resistance (multi-timeframe)
- Compute support/resistance across:
  - daily
  - 1h
  - 5m
Methods:
- pivot-based (swing pivots)
- optional: rolling high/low zones
- store multiple levels with confidence/strength score

F) Swing High/Low on 5-min
- Identify swing highs/lows using pivot window (e.g., left=3 right=3)
- Display most recent swing high and swing low + timestamps
- Use these swings to derive S/R levels

Build a grid/table with one row per ticker showing:
- current price

- EMA crossover state + last crossover time
- VWAP position (above/below) + distance + toward/away indicator
- daily trend (bull/bear/neutral)
- session POC value

Charts:
- TradingView Lightweight Charts candlestick (5m)
- Overlays: EMA9, EMA21, VWAP, S/R lines, swing markers
- A per-ticker detail view when clicking a row

Alerts panel:
- stream alerts for events:
  - EMA cross
  - VWAP reclaim/breakdown
  - break of support/resistance
  - new swing high/low

========================================================
Every 5 minutes, for EACH ticker, run an AI evaluation that predicts near-term direction for the next 1–3 five-minute candles.

- The AI is NOT allowed to use external facts, news, or assumptions.
- The AI must ONLY use the computed feature set from this app (EMA, VWAP, trend, POC, S/R, swings, recent returns/volatility).

- The AI must include the exact feature values it used.
- If confidence is below a threshold or features are insufficient/inconsistent, output ABSTAIN.
